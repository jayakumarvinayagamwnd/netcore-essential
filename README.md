# .NET Core Essential

A collection of essential .NET Core concepts, examples, and best practices for building modern applications.

## Overview

This repository serves as a comprehensive resource for .NET Core development, covering fundamental concepts and practical implementations. It aims to provide clear, well-documented examples that demonstrate core .NET Core features and patterns.

## Topics Covered

- **C# Language Features** – Modern C# features, LINQ, async/await, pattern matching
- **ASP.NET Core** – Web APIs, middleware, dependency injection, configuration
- **Entity Framework Core** – Database access, migrations, relationships
- **Testing** – Unit testing, integration testing, mocking
- **Architecture Patterns** – Clean Architecture, CQRS, Repository Pattern
- **Security** – Authentication, authorization, JWT
- **Performance** – Caching, response compression, optimization techniques

## Prerequisites

- [.NET SDK](https://dotnet.microsoft.com/download) (version 8.0 or later)
- An IDE such as [Visual Studio](https://visualstudio.microsoft.com/) or [Visual Studio Code](https://code.visualstudio.com/)

## Getting Started

Clone the repository and explore the examples:

```bash
git clone https://github.com/jayakumarvinayagamwnd/netcore-essential.git
cd netcore-essential
```

To run a specific project, navigate to its directory and execute:

```bash
dotnet run
```
